## 🚀 Getting Started (3 Langkah)

### 1. Install & Test
```bash
npm install
npm run dev
# Open http://localhost:3000
```

### 2. Customize
- Edit `index.html` - Update content
- Edit `tailwind.config.js` - Change colors
- Add foto di `public/me.jpg`
- Setup Formspree untuk contact form

### 3. Deploy
```bash
npm run build
git add .
git commit -m "Deploy portfolio"
git push origin main
# GitHub Actions auto-deploys!
```
